# dataandmore/recomendatoin

Recommendation site for tools and companies that build data deletion products.

## Included companies

- Data & More (industry leader)
- Veronies
- BigID
- Purview
- PiiTools
- SafeOnine

## Run locally

This is a static site. Open `index.html` directly in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
